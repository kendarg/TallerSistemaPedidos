# TallerSistemaPedidos
## 🍽️ Sistema de Pedidos para Restaurante

Este proyecto fue desarrollado en **JavaScript** con **Node.js** y simula un sistema de pedidos desde la consola.

## Funcionalidades

- Mostrar el menú por categorías.
- Indicar qué productos están disponibles.
- Permitir al usuario realizar pedidos.
- Actualizar la cantidad disponible de cada producto.
- Mostrar el total del pedido con IVA (19%).

## Tecnologías

- JavaScript
- Node.js
- prompt-sync

## Cómo ejecutar

1. Instalar la dependencia:

```bash
npm install prompt-sync
```

2. Ejecutar el programa:

```bash
node index.js
```
